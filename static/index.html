Rig Chat 嵌入式组件示例

简介

Rig Chat 可以轻松嵌入到任何网站中，只需添加一行 JavaScript 代码。 聊天组件将作为浮动窗口出现在您的页面上，用户可以随时打开或关闭它。

基本嵌入方法

将以下代码添加到您的 HTML 页面中：

<!-- 1. 引入 RigChat 脚本 -->
<script src="https://your-server.com/chatbot.js"></script>

<!-- 2. 创建聊天组件实例 -->
<script>
    const chatbot = new RigChat({
        apiBase: "https://your-api-server.com",  // API 服务器地址
        title: "客服助手",                       // 聊天窗口标题
        welcomeMessage: "您好！我是您的智能助手", // 欢迎消息
        buttonIcon: "ai",                    // 按钮图标
        theme: "light",                        // 主题颜色
        position: "right",                     // 窗口位置
        placeholder: "请输入您的问题..."         // 输入框提示文字
        containerId: "rig-chat-container"       // 容器ID
        defaultWidth: 450,                     // 默认宽度
        defaultHeight: 550                     // 默认高度
    });
</script>

所有配置选项都是可选的。如果不提供，将使用默认值。

配置选项

在创建 RigChat 实例时，您可以传入以下配置选项：

选项 默认值 说明
apiBase 当前域名 Rig API 的基础 URL
theme "light" 主题颜色 ("light" 或 "dark")
position "right" 聊天窗口位置 ("right" 或 "left")
welcomeMessage "Welcome to Rig Assistant! How can I help you today?" 初始欢迎消息
title "Rig Assistant" 聊天窗口标题
buttonIcon "ai" 聊天按钮图标名称
placeholder "Type your message..." 输入框提示文字
containerId "rig-chat-container" 聊天组件容器的 ID
defaultWidth 450 默认宽度
defaultHeight 550 默认高度

示例：使用自定义配置创建聊天组件

const chatbot = new RigChat({
    apiBase: "https://api.example.com",
    theme: "dark",
    position: "left",
    title: "智能客服",
    welcomeMessage: "您好！请问有什么可以帮您？",
    buttonIcon: "ai",
    placeholder: "输入您的问题..."
});

自定义样式

如果您需要自定义聊天组件的样式，可以通过 CSS 变量在您的网站样式表中进行覆盖： 

/* 示例：自定义主题色 */
:root {
    --bg-accent: #4f46e5; /* 主题色 */
    --user-bubble: #4f46e5; /* 用户消息气泡颜色 */
}

/* 示例：自定义聊天按钮大小 */
#rig-chat-button {
    width: 56px !important;
    height: 56px !important;
}

API 集成

Rig Chat 组件默认使用以下 API 端点：

  • /api/chat - 处理聊天请求
  • /api/history/{user_id} - 获取聊天历史

确保您的后端服务提供这些 API 端点，或者通过 RIG_CHAT_API_BASE 配置指向正确的 API 服务器。

本地化

您可以通过修改 welcomeMessage , title , placeholder 来本地化聊天组件的文本。 对于更高级的本地化需求，您可以修改 chatbot.js 文件，替换所有可见文本。
